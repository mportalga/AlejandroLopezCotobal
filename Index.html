Ordenador adaptado para personas con problemas



Según un estudio realizado por GESOP, tres de cada cuatro personas con discapacidad física, visual o auditiva encuestadas disponen de ordenador para el uso personal. Pero sólo una minoría utiliza un ordenador con adaptaciones, el 5% en el caso del ordenador de sobremesa y el 4% para los portátiles. La mayor parte de las adaptaciones son de control físico especial -41%-, como por ejemplo: aplicaciones para escribir con la mirada, conmutadores, joysticks, trackball, micrófono, sensor ocular, etc. También son frecuentes otros adapatacions como el control por voz -29%- o la ampliación visual -20%-. Finalmente, los datos muestran que a medida que aumenta el grado de discapacidad también se incrementa el uso diario del ordenador (esté o no adaptado).

    Teclado

    Este ordenador tendra un teclado adaptado para personas con baja visibilidad, tendrá el alfabeto en braille y otras funciones, como permitir escribir con la mirada o la función dictar. Con el teclado braile de TalkBack:

  • Puedes utilizar 6 en tu pantalla para escribir en alfabeto braille de seis puntos.
  • Está disponible en árabe, español, inglés unificado y polaco.

    • Importante: Para usar el teclado braille de TalkBack, debes activar TalkBack y desactivar la ampliación.

    Pantalla

    La herramienta que han creado, llamada 4all permite manejar una pantalla táctil con cualquier puntero o pulsador adaptado a la discapacidad. El producto es una caja de un tamaño inferior a 5 centímetros y una aplicación que permite conectar cualquier dispositivo tecnológico que utilice el sistema Android de Google a los pulsadores que utilizan las personas con discapacidad física, permitiéndoles usarlos de forma autónoma. Con este sistema, se crea un eje horizontal y otro vertical sobre la pantalla que el usuario puede mover con su ratón o puntero para señalar la opción deseada dentro de su teléfono y navegar como lo haría cualquier otro usuario con sus dedos y además, no sólo les permite pulsar sino que también les permite la opción de deslizar para moverse por la pantalla, como por ejemplo para recorrer una conversación en una aplicación de mensajería.
